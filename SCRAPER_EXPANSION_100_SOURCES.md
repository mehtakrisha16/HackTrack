## üìä SCRAPING APPROACH

### For Each Source:
1. **Identify Scraping Method:**
   - Direct HTML scraping (Puppeteer/Cheerio)
   - API endpoints (if available)
   - RSS feeds
   - Structured data (JSON-LD, Schema.org)

2. **Data Fields to Extract:**
   - Title
   - Company/Organizer
   - Description
   - Location (City, State, Country, Remote)
   - Date (Posted, Deadline, Event Date)
   - Type (Hackathon, Internship, Event, Job)
   - Category/Domain
   - Tags/Technologies
   - Application Link
   - Salary/Prize (if applicable)

3. **Quality Checks:**
   - Deduplicate by title + company + location
   - Validate URLs
   - Check for expired opportunities
   - Verify data completeness

4. **Error Handling:**
   - Retry failed requests (3 attempts)
   - Log errors with source identification
   - Continue on individual source failure
   - Report success/failure metrics
